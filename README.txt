A program to convert an SMS backup into an html web page for easier viewing.

SETUP:
  To be used in conjunction with Android app SMS Backup & Restore by Carbonite. 
  In app, back up a single conversation, and change the resulting file name to "sms.txt".
  Place "sms.txt" in same directory as "run.bat".
  Upon executing "run.bat", "sms.txt" will be converted to an html file called "output.html" and opened immediately.
